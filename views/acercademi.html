Mi nombre es Maly Puerto y soy una desarrolladora de fullstack muy entusiasta. Mi primera página la realicé durante mi estudio universitario pero fue hasta que me capacité con BICTIA que aprendí a programar para desarrollo de páginas web. Gracias a ellos se desarrollar en html / css / javascript con el framework de Bootstrap, y con las herramientas Node y Express. También aprendí a combinarlo con bases de datos MySQL y MongoDB. Adicionalmente, trabajé por un periodo con una startup con Materialize y React. También, realicé los cursos de la beca del Ministerio de Tecnología que incluyeron análisis de datos y Python. Para el trabajo en grupo, manejo GIT y la metodología ágil de gestión SCRUM.

Soy tranquila, comunicativa y proactiva, siempre busco mejorar mis habilidades y enfrentar nuevos retos. Estoy dispuesta a asumir proyectos nuevos e interesantes y a trabajar con otros dentro de un equipo de desarrollo y multidisciplinario.
